If you want to follow along, you can proceed to these setup actions:

- Interpreter: Python 3.12.8

- Open a new folder, e.g., "chatbot_poc", and access it through a Python editor, e.g., visual studio code

- Inside the folder, create and activate a Python virtual environment using the following:

     - python -m venv .venv (this creates a virtual environment named ".venv")

     - Activate it by running .venv\Scripts\activate

The above is for Windows users. If you are on Linux or Mac, use the corresponding commands.

- Make a requirements.txt file and paste the following:


langchain-core==0.3.33
langchain-openai==0.3.3
langgraph==0.2.69
matplotlib==3.10.0
openai==1.61.1
colorama==0.4.6
python-dotenv==1.0.1


- Then run pip install -r requirements.txt

- Also, make a .env file and paste your OpenAI key inside:

 

OPENAI_API_KEY = 'your secret OpenAI key goes here'


If you do not have such a key, there are instructions to run the code with ollama, e.g., locally.


And that's it!
