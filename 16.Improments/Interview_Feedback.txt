3. Python code to find top 3 salaries for each category in a dataframe.
   ```python
   import pandas as pd

   # Sample DataFrame
   data = {
       'Category': ['A', 'A', 'A', 'B', 'B', 'C', 'C', 'C'],
       'Salary': [50000, 60000, 55000, 70000, 80000, 90000, 85000, 95000]
   }
   df = pd.DataFrame(data)

   # Finding top 3 salaries for each category
   top_salaries = df.groupby('Category')['Salary'].nlargest(3).reset_index()
   top_salaries = top_salaries.drop('level_1', axis=1)

   print(top_salaries)
   ```

4. difference between adboost, xgboost and gradient boost.
   - Both AdaBoost and Gradient Boosting are ensemble learning techniques that combine multiple weak learners to create a strong learner. The key difference lies in how they build the model:
     - AdaBoost focuses on adjusting the weights of incorrectly classified instances, giving more importance to those instances in subsequent models.
     - Gradient Boosting, on the other hand, builds models sequentially by optimizing a loss function, where each new model attempts to correct the errors made by the previous models.
   - XGBoost (Extreme Gradient Boosting) is an optimized implementation of Gradient Boosting that includes additional features such as regularization, parallel processing, and handling missing values, making it faster and often more accurate than standard Gradient Boosting.
