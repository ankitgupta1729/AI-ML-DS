2 Annotated layout (conversation screen)

  • 1
    Header — GO brand mark, app name + tagline, live status (indexed chunks & model), theme toggle, clear chat.
  • 2
    Sidebar — navigate Chat · Mock Test · Flashcards · Planner · Dashboard · Daily (with a due-cards badge).
  • 3
    User message — right-aligned red gradient bubble; shows attached image thumbnails / file chips.
  • 4
    Assistant message — GO avatar + bubble; Markdown, KaTeX maths, code; streams token-by-token; grounding-confidence meter.
  • 5
    Source cards — expandable: file, page/slide, subject tag, % match.
  • 6
    Action bar — copy · 👍 · 👎 · ↻ regenerate · 🔊 read-aloud.
  • 7
    Follow-up chips — contextual one-click prompts.
  • 8
    Composer — auto-grow textarea, 📎 attach, ✏️ sketch, 🎙️ voice, Socratic toggle & language select; Enter sends.
