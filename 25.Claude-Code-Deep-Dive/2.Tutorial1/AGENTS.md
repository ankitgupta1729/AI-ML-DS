# Repository Guidelines

## Project Overview

Tutorial project for learning **Claude Code**, Anthropic's agentic coding CLI tool. Part of a Claude Code Deep Dive course.

## Project Structure & Module Organization

Currently a single-file starter project. `Notes.md` contains step-by-step instructions for installing and using Claude Code.

## Commit & Pull Request Guidelines

Commits use short descriptive messages (e.g., `added content`). No PR template is configured.
