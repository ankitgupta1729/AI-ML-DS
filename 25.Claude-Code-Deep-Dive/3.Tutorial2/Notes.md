6. To save credits in claude so that it will not say like use after few hours.

- Don't have long chats, instead start fresh chats because in a long chat, it always takes all the previous context for a new message and it takes a lot more tokens. So, you can do as put `summarize the context` in chat and copy the context and start a new chat.
- You can edit the existing prompt if claude does not give the correct response. So, instead of giving a new prompt in later chat conversation, you can edit the existing prompt.
- Instead of asking one question at a time, combine your questions and ask them at once.
- Utilize claude projects by putting all the files in one place instead of uploading files again and again.
- Setup the permanent memory in settings. So, got `Settings` and then `Capabilities` tab and turn on the `Generate memory from chat history`.
- Use the right model. `Haiku` and `Sonnets` are good for writing and simple tasks but for deeper and long research tasks, opus is the way to go. `Opus` is a better model but also it takes more tokens/credits. 
- Work in sessions like morning, afternoon, evening etc. becuase it has 5 hours window.
