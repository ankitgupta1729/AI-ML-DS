6. Claude Skills:

You can go to Settings and then Capabilities and you can browse skills and create new skills also. 

For example, you can create a skill by first give access claude to a skill as `slack-gif-creater` from settings--> capabilities --> browse skills --> slack-gif-creater and then going to claude chat and write:

`generate an animated gif of drinking coffee for slack`

Now, while generating response, it will show first like `Read the Slack GIF creator skill`.

Now, if you have to build a new skill, first you can see the examples by clicking on upload an skill option and there in an option below to see examples and it will  navigate to `https://github.com/anthropics/skills/blob/main/skills/algorithmic-art/SKILL.md` and there are many other skills. 

There are various options like create skills with claude or give skill instructions or upload zip, .skill or .md files.

`skill-creator` is also a skill that you can use to create new skills.

First we create an skill using `Create with Claude` option. 

A. First write the prompt as `Let's create a skill together using your skill-creator skill. First ask me what the skill should do.`

B. Then write the prompt as `I want something where I drop my csv file and ask to create dashboard then it creates industry level dashboard with all business related ROI metrices.`

C. Then answer few questions and it will generate dashboard related files and `skill.md` file.

You can use this md file later like you can add it as prompt file and chat with claude to do more things in the dashboard.
